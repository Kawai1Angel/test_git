​			当 $n > 1$ 时，有 $T(n) = 1 + 1 + \sum\limits_{k = 1}^{n - 1} (T(k) + 1) + 1$。			于是我们得到 $T(n) = \sum\limits_{k = 1}^{n - 1}T(k) + n + 2$。

​			同理可得 $T(n - 1) = \sum\limits_{k = 1}^{n - 2}T(k) + n + 1$​。

​			即 $T(n) + 1 = 2 (T(n - 1) + 1)$。

​		最紧时间复杂度：

​			当 $n = 1$ 时，有 $T(1) = 1 + 1 + 1 = 3$。
